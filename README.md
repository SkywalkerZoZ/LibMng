# LibMng



## Environment
java 11.0.21 2023-10-17 LTS



## Project Structure



**src/main/java/com/xdc5/libmng**



| 目录        | 作用                              |
| ----------- | --------------------------------- |
| /controller | 存放controller层相关类，处理请求  |
| /entity     | 存放实体类                        |
| /mapper     | 存放dao层相关类，映射数据库       |
| /service    | 存放service层相关类，处理业务逻辑 |
| /utils      | 存放工具类                        |
